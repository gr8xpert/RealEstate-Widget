General Questions

Smart Property Widget is a powerful property listing solution that allows real estate agencies and property websites to display their property listings beautifully. It provides search filters, property grids, carousels, detailed property pages, and many more features to create a professional property website.
Getting started is simple:
  • WordPress users: Install and activate the Smart Property Widget plugin from your WordPress admin panel
  • Other platforms: Add a small script to your website and start using the widget components
Once set up, your properties will automatically sync and display on your website.
No technical knowledge is required for basic usage. The widget comes with ready-made templates that you can simply add to your pages. For WordPress users, you can use page builders like Elementor or Divi to drag and drop components. Advanced customization options are available for those who want more control.
Yes, your data is secure. All data transfers are encrypted using HTTPS. Your property data is synced from your existing property management system and displayed securely on your website. We do not store sensitive client information and follow industry-standard security practices.

Features

Smart Property Widget offers comprehensive search filters including:
  • Location: Search by area, city, or region
  • Property Type: Villa, apartment, townhouse, penthouse, etc.
  • Listing Type: For sale, long-term rental, short-term rental
  • Price Range: Minimum and maximum price
  • Bedrooms & Bathrooms: Filter by number of rooms
  • Size: Built area and plot size
  • Features: Pool, garden, garage, sea views, etc.
  • Reference Number: Direct property lookup
Yes! The widget includes a wishlist feature that allows visitors to save properties they're interested in. Saved properties are stored in the browser, so visitors can return later to view their favorites. You can also add a wishlist page to your website where visitors can see all their saved properties at once.
Yes, Smart Property Widget supports 16 languages including English, Spanish, German, French, Dutch, Portuguese, Italian, Russian, Swedish, Danish, Norwegian, Finnish, Polish, Czech, Romanian, and Arabic. The widget automatically detects the visitor's language or you can set a specific language for your website.
Yes, the widget includes an interactive map view feature. Visitors can browse properties on a map, see property markers with price tags, and click on markers to view property details. The map integrates with Google Maps to show property locations accurately.
Yes, the widget offers multiple display options:
  • Grid View: Display properties in a responsive grid (1-4 columns)
  • List View: Show properties in a detailed list format
  • Carousel: Sliding property showcase for featured listings
  • Map View: Interactive map with property markers
There are also 12 different card templates to choose from, each with a unique design style.
Yes! If your properties have video tours (YouTube, Vimeo) or virtual tours (Matterport, etc.), these will automatically display on the property detail page. Visitors can watch videos and explore virtual tours directly on your website without leaving the page.
Yes, each property detail page includes an inquiry form. When visitors submit an inquiry, it is sent directly to your email address. You can customize the thank you message shown after submission or redirect visitors to a custom thank you page.
