Property Carousel - Template 01

Center Focus Carousel with 5 visible cards, size/opacity gradient effect

Live Demo - Basic Usage

Basic Usage

<!-- Basic carousel - loads properties from API --> <div class="rs_property_carousel"></div>

With Options

<!-- Carousel with all options --> <div class="rs_property_carousel" data-rs-limit="10" data-rs-autoplay="true" data-rs-interval="5000" data-rs-featured="true"></div>

Live Demo - Auto-play Enabled (Featured Properties)

Data Attributes Reference

Attribute Default Description
data-rs-limit 10 Maximum number of properties to load
data-rs-autoplay false Enable auto-play rotation
data-rs-interval 5000 Auto-play interval in milliseconds
data-rs-featured false Only show featured properties
data-rs-variation 1 Template variation (future: 2, 3, 4)

Features

  • 5 visible cards with center focus effect (largest card in center)
  • Size and opacity gradient from center to edges
  • Smooth 1s transitions on all properties
  • Touch/swipe support for mobile devices
  • Keyboard navigation (arrow keys)
  • Auto-play with pause on hover
  • Dot indicators for navigation
  • Infinite loop navigation
  • Responsive design (adapts to tablet/mobile)
  • Simplified property cards (image, title, beds/baths, price)
