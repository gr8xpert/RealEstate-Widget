Property Carousel - Template 02

3D Perspective Carousel with rotation, depth, and blur effects

Live Demo - 3D Perspective

Usage

<!-- 3D Perspective Carousel --> <div class="rs_property_carousel" data-rs-variation="2" data-rs-limit="10"></div>

Features

  • 3D perspective rotation effect
  • Cards scale and translate in Z-axis (depth)
  • Blur effect on non-active cards
  • Smooth 0.4s transitions
  • Only center card is clickable
  • Touch/swipe support
  • Keyboard navigation (arrow keys)
  • Responsive design
