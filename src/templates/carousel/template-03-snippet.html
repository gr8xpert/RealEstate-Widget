Property Carousel - Template 03

Coverflow Carousel with rotated cards facing center and greyscale effect

Live Demo - Coverflow

Usage

<!-- Coverflow Carousel --> <div class="rs_property_carousel" data-rs-variation="3" data-rs-limit="10"></div>

Features

  • Coverflow 3D rotation effect (cards face center)
  • Greyscale/desaturation on side cards
  • Large rounded corners (20px border-radius)
  • Smooth 0.6s transitions
  • Only center card is clickable
  • Touch/swipe support
  • Keyboard navigation (arrow keys)
  • Infinite loop navigation
  • Responsive design
